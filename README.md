# Open Horizon VR Open Source Library.
The Open Horizon VR Open Source Library is to be an open source VR library for any developers looking to use open sourced assets used in the Open Horizon game on Roblox. Being developed by @AptAlbatross & @Desty_e on Roblox.

Files here will be more regularly updated and contain the edited version of the RawVR client used in Open Horizon.
